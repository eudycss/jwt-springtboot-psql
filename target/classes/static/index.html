API de Autenticación y Autorización JWT

Esta API proporciona funcionalidades para autenticación y autorización de usuarios usando JWT (JSON Web Tokens) en Spring Boot.

Nota: Para acceder a la documentación completa de la API, visita /swagger-ui.html

Manejo de Contraseñas

La API acepta contraseñas tanto en formato de texto plano como en Base64:

Texto Plano

Puedes enviar tu contraseña directamente sin codificar:

{
  "username": "docente1",
  "password": "miContraseña123",
  "roles": ["docente"]
}

Base64 (Mayor seguridad)

También puedes enviar tu contraseña codificada en Base64:

{
  "username": "docente1",
  "password": "bWlDb250cmFzZcOxYTEyMw==",
  "roles": ["docente"]
}

La API detectará automáticamente si la contraseña está en Base64 o en texto plano.

Importante: Para mayor seguridad en entornos de producción, se recomienda:
  • Usar siempre HTTPS
  • Enviar las contraseñas en formato Base64
  • Implementar mecanismos adicionales como OAuth2

Endpoints Principales

Endpoint Método Descripción Acceso
/api/auth/register POST Registrar un nuevo usuario Público
/api/auth/login POST Iniciar sesión y obtener token JWT Público
/api/docente/** GET Acceso a recursos de docentes DOCENTE, RECTOR
/api/rector/** GET Acceso a recursos de rectores Solo RECTOR
/api/users/** GET, PUT, DELETE Gestión de usuarios Solo RECTOR

Ejemplo de Codificación en Base64

JavaScript (Navegador)

// Para codificar
const password = "miContraseña123";
const base64Password = btoa(password);
console.log(base64Password); // "bWlDb250cmFzZcOxYTEyMw=="

// Para decodificar
const originalPassword = atob(base64Password);
console.log(originalPassword); // "miContraseña123"

JavaScript (Node.js)

// Para codificar
const password = "miContraseña123";
const base64Password = Buffer.from(password).toString('base64');
console.log(base64Password);

// Para decodificar
const originalPassword = Buffer.from(base64Password, 'base64').toString();
console.log(originalPassword);

Java

// Para codificar
String password = "miContraseña123";
String base64Password = Base64.getEncoder().encodeToString(password.getBytes());
System.out.println(base64Password);

// Para decodificar
byte[] decodedBytes = Base64.getDecoder().decode(base64Password);
String originalPassword = new String(decodedBytes);
System.out.println(originalPassword);
